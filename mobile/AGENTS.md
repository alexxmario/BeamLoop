# Expo SDK version

This app is pinned to **Expo SDK 54** (`expo: ~54.0.0`, `react-native: 0.81.5`).
Read the exact versioned docs at https://docs.expo.dev/versions/v54.0.0/ before
writing any code.

Do NOT bump the SDK without rechecking the target device's Expo Go — the pin to
54 is deliberate (the iPhone used for testing tops out at SDK 54).
